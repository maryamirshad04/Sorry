# 💗 Apology Website

A cute personalized apology website.

## Features

- 💕 Heart cursor throughout
- 🎮 "No" button runs away + grows "Yes" button  
- 📸 Coupon 1: Upload couple photo + music player with your song
- 🎡 Coupon 2: Spin wheel — always lands on 100% love!
- 🌸 Coupon 3: Bouquet with clickable flowers and their meanings
- 💌 Coupon 4: Feedback form that copies to clipboard (no database needed!)
- ✨ Floating hearts background
- 📱 Fully mobile responsive
