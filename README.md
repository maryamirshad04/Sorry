# 💗 Apology Website

A cute personalized apology website built with Next.js, ready to deploy on Vercel!

## Setup

```bash
npm install
npm run dev
```

## 🐱 Adding Cat GIFs

Place your cat GIFs in the `/public` folder:
- `public/cat-happy.gif` — shown when "yes" is clicked
- `public/cat-sad.gif` — shown when "no" is clicked

If no GIFs are provided, the site falls back to emoji (😸 and 😿).

**Recommended free cat GIFs:**
- Happy cat: Search "happy cat pixel gif" on Tenor or Giphy
- Sad cat: Search "sad cat crying gif" on Tenor or Giphy

## Deploy to Vercel

1. Push to GitHub
2. Connect repo on vercel.com
3. Deploy! (No environment variables needed)

## Features

- 💕 Heart cursor throughout
- 🎮 "No" button runs away + grows "Yes" button  
- 📸 Coupon 1: Upload couple photo + music player with your song
- 🎡 Coupon 2: Spin wheel — always lands on 100% love!
- 🌸 Coupon 3: Bouquet with clickable flowers and their meanings
- 💌 Coupon 4: Feedback form that copies to clipboard (no database needed!)
- ✨ Floating hearts background
- 📱 Fully mobile responsive
